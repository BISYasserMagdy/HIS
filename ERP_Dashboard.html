💾 Backup Database

Create a complete backup of the pharmacy database including all medicines, sales, purchases, suppliers, and user data. The backup file will be downloaded as a compressed SQL file.

Last backup: Not recorded

📋 Backup Information

  • ✔ All medicine inventory records
  • ✔ Sales and transaction history
  • ✔ Purchase orders and supplier data
  • ✔ Prescriptions and returns
  • ✔ User accounts (passwords not included)

🔄 Restore Database

Restore the pharmacy database from a previously downloaded backup file (.sql). Warning: This will overwrite current data with the backup. Only use files generated by this system.

⚠️ Important Notes

  • Only upload backup files generated by this ERP system.
  • Current data will be replaced with the backup content.
  • A confirmation prompt will appear before restoring.
  • It is recommended to take a fresh backup before restoring.

⚠️ Factory Reset

This will permanently delete all data from the pharmacy system — medicines, sales, purchases, prescriptions, suppliers, and all records. This action is irreversible.

To protect your data, a verification email will be sent to the Manager's registered Gmail. The reset will not occur until you click the confirmation link in that email.
